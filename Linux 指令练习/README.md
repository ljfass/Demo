# Linux 指令练习
                                 文件                  文件夹
    新建：                       touch file                                                mkdir  dir
    赋值：                       cp file1 file2                                           cp -r dir1 dir2
    剪切：                       mv file1 file2                                           mv dir1 dir2
    删除：                        rm file                                                    rm -r dir
    进入：                        vi/cat/head/tail/more file                      cd dir
    文件修改：                i插入 esc退出 :q!推出  ：wq!保存退出  ctrl + insert 复制           shift + insert 粘贴
